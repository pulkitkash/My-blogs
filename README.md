## 🚀 How to Use This Repo

### Read Online
Simply click any blog link in the **Blog Index** above GitHub renders Markdown beautifully.

### Read Locally
```bash
# Clone the repository
git clone https://github.com/your-username/tech-blogs.git

# Navigate to a blog
cd tech-blogs/web-scraping/
```

### Download PDF Version
Each blog folder contains a compiled `.pdf` file. Click it on GitHub and hit **Download**.

### Run Code Examples
```bash
# Install dependencies for web scraping blogs
pip install requests beautifulsoup4 scrapy selenium pandas

# Run example scripts
python examples/basic_scraper.py
```
